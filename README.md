# 🌱 ENKOMOS-Fertilizer

**Indigenous wisdom • Traditional methods • Modern organic upgrades**

A mobile-ready, offline-capable web application that helps farmers, home growers, and agricultural enterprises calculate organic fertilizer recipes, costs, and production methods based on traditional knowledge and modern organic substitutes.

---

## 🎯 Purpose

ENKOMOS-Fertilizer bridges the gap between ancient farming wisdom and contemporary organic agriculture. It provides:

- Step-by-step production processes for traditional fertilizers
- Cost comparison (traditional vs modern organic vs chemical)
- Dynamic kit recommendations based on **space type** (open field, shaded, greenhouse) and **grower type** (corporate, medium, home)
- Downloadable recipe reports

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🌾 **5 Fertilizer Types** | Jeevamrut, Vermicompost, Panchagavya, Neem Cake, Farmyard Compost |
| 🌿 **Space Selection** | Open field / Shaded / Greenhouse – adjusts application advice |
| 👨‍🌾 **Grower Tiers** | Corporate (≥1 ha) / Medium (0.1–1 ha) / Home grower (non‑commercial) |
| 🧪 **Ingredient Options** | Traditional local ingredients or modern certified organic substitutes |
| 💰 **Cost Comparison** | Shows cost per unit and total, plus chemical alternative for reference |
| 📥 **Download Recipe** | Save the full recipe as a .txt file for offline use |
| 📱 **Mobile Ready** | Works on any smartphone without internet after first load |

## 🗂️ Fertilizer Database

All recipes are sourced from indigenous knowledge across South Asia (India, Nepal, Bangladesh) and refined with modern organic inputs. Each entry includes:

- Ingredients (traditional & modern)
- Production process
- Application method
- Monitoring indicators
- Required equipment
- Time to maturity
- Minimum and bulk batch sizes
- Cost per unit (USD)

- Responsive design (mobile first)
- Works offline after first visit (cached)
- No server, no database – runs entirely in the browser
